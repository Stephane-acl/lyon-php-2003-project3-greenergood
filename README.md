# The Greener Good - Gare centrale
### Prerequisites

1. Check composer is installed
2. Check yarn & node are installed

## Deployment

* Run `composer install`
* Run `yarn install`
* Change database infos in .env (line 27)
* Change mailer DSN in .env (line 31)
